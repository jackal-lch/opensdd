# Rust Test Patterns

Test patterns for Rust projects using the built-in test framework.

## Test Location

- **Unit tests**: In same file as code, inside `#[cfg(test)]` module
- **Integration tests**: In `tests/` directory at crate root

## Basic Structure

```rust
// In src/component_name.rs

// ... implementation code ...

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn test_function_name_happy_path() {
        // Arrange
        let component = ComponentName::new();
        let input = InputType { field: "value".into() };

        // Act
        let result = component.function_name(input);

        // Assert
        assert!(result.is_ok());
        let output = result.unwrap();
        assert_eq!(output.field, expected);
    }

    #[test]
    fn test_function_name_error_case() {
        // Arrange
        let component = ComponentName::new();
        let input = InputType { field: "invalid".into() };

        // Act
        let result = component.function_name(input);

        // Assert
        assert!(result.is_err());
        match result {
            Err(ErrorType::InvalidInput) => (),
            _ => panic!("Expected InvalidInput error"),
        }
    }

    #[test]
    #[should_panic(expected = "cannot be empty")]
    fn test_function_name_panics_on_empty() {
        let component = ComponentName::new();
        component.function_name(InputType::default());
    }
}
```

## Integration Tests

```rust
// tests/component_name_test.rs

//! Integration tests for ComponentName
//!
//! Generated by /opensdd:tdd-spec from spec.yaml
//! Component: ComponentName

use my_crate::{ComponentName, InputType, OutputType, ErrorType};

#[test]
fn test_component_integration() {
    let component = ComponentName::new();
    let input = InputType::new("value");

    let result = component.function_name(input);

    assert!(result.is_ok());
}
```

## Assertions

```rust
// Equality
assert_eq!(actual, expected);
assert_ne!(actual, unexpected);

// Boolean
assert!(condition);
assert!(!condition);

// Option
assert!(option.is_some());
assert!(option.is_none());
assert_eq!(option, Some(expected));

// Result
assert!(result.is_ok());
assert!(result.is_err());
let value = result.unwrap();
let error = result.unwrap_err();

// Pattern matching
assert!(matches!(value, Pattern::Variant));
assert!(matches!(result, Ok(OutputType { field: "expected", .. })));
assert!(matches!(result, Err(ErrorType::InvalidInput)));

// Panic
#[test]
#[should_panic]
fn test_panics() {
    function_that_panics();
}

#[test]
#[should_panic(expected = "error message")]
fn test_panics_with_message() {
    function_that_panics();
}
```

## Test Attributes

```rust
// Basic test
#[test]
fn test_name() {}

// Ignored test (skip by default)
#[test]
#[ignore]
fn slow_test() {}

// Run ignored: cargo test -- --ignored

// Expected panic
#[test]
#[should_panic]
fn test_panics() {}

// Expected panic with message
#[test]
#[should_panic(expected = "message")]
fn test_panics_with_message() {}
```

## Result-Returning Tests

```rust
#[test]
fn test_with_result() -> Result<(), Box<dyn std::error::Error>> {
    let component = ComponentName::new();
    let result = component.function_name(input)?;

    assert_eq!(result.field, expected);
    Ok(())
}
```

## Async Tests (tokio)

```rust
#[tokio::test]
async fn test_async_function() {
    let component = ComponentName::new();
    let result = component.async_function(input).await;

    assert!(result.is_ok());
}

#[tokio::test]
async fn test_async_error() {
    let component = ComponentName::new();
    let result = component.async_function(bad_input).await;

    assert!(matches!(result, Err(ErrorType::InvalidInput)));
}
```

## Mocking with mockall

```rust
use mockall::predicate::*;
use mockall::mock;

// Define mock trait
mock! {
    pub Dependency {}

    impl DependencyTrait for Dependency {
        fn method_name(&self, input: InputType) -> Result<OutputType, ErrorType>;
    }
}

#[test]
fn test_with_mock() {
    let mut mock_dep = MockDependency::new();
    mock_dep
        .expect_method_name()
        .with(eq(expected_input))
        .times(1)
        .returning(|_| Ok(OutputType::default()));

    let component = ComponentName::with_dependency(mock_dep);
    let result = component.function_name(input);

    assert!(result.is_ok());
}
```

## Event Testing

```rust
use std::sync::{Arc, Mutex};

#[test]
fn test_emits_event() {
    let events: Arc<Mutex<Vec<EventType>>> = Arc::new(Mutex::new(Vec::new()));
    let events_clone = events.clone();

    let component = ComponentName::new();
    component.on_event(move |event| {
        events_clone.lock().unwrap().push(event);
    });

    component.trigger_action();

    let captured = events.lock().unwrap();
    assert_eq!(captured.len(), 1);
    assert_eq!(captured[0].field, expected);
}
```

## Test Organization with Modules

```rust
#[cfg(test)]
mod tests {
    use super::*;

    mod function_name {
        use super::*;

        #[test]
        fn happy_path() {
            // ...
        }

        #[test]
        fn error_case() {
            // ...
        }

        mod edge_cases {
            use super::*;

            #[test]
            fn handles_empty_input() {
                // ...
            }

            #[test]
            fn handles_max_value() {
                // ...
            }
        }
    }

    mod events {
        use super::*;

        #[test]
        fn emits_event_on_success() {
            // ...
        }
    }
}
```

## Property-Based Testing (proptest)

```rust
use proptest::prelude::*;

proptest! {
    #[test]
    fn test_roundtrip(input in ".*") {
        let encoded = encode(&input);
        let decoded = decode(&encoded).unwrap();
        prop_assert_eq!(input, decoded);
    }

    #[test]
    fn test_never_panics(n in 0..1000i32) {
        let _ = potentially_dangerous_function(n);
    }
}
```

## Test Data Factories

```rust
impl Credentials {
    #[cfg(test)]
    pub fn test_default() -> Self {
        Self {
            email: "test@example.com".into(),
            password: "TestPassword123".into(),
        }
    }

    #[cfg(test)]
    pub fn with_email(mut self, email: impl Into<String>) -> Self {
        self.email = email.into();
        self
    }
}

// Usage
#[test]
fn test_authenticates_valid_credentials() {
    let credentials = Credentials::test_default();
    // ...
}

#[test]
fn test_rejects_invalid_email() {
    let credentials = Credentials::test_default()
        .with_email("invalid");
    // ...
}
```

## Running Tests

```bash
# Run all tests
cargo test

# Run specific test
cargo test test_function_name

# Run tests in specific module
cargo test tests::function_name

# Show output
cargo test -- --nocapture

# Run ignored tests
cargo test -- --ignored

# Run with verbose output
cargo test -- --show-output
```
