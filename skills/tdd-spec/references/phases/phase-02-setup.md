<step n="2" name="create_test_file">
Create test file with ALL tests SKIPPED.

**Key: Tests are skipped/disabled initially.** We'll enable them one-by-one in Phase 3.

**Reference patterns:** See `references/patterns/{language}.md`

**Skip mechanisms:**

→ See: `lookup.md § Skip Syntax`

**Generate test file structure:**

```typescript
// Example: TypeScript/Vitest

// Generated by /opensdd:tdd-spec from spec.yaml
// Component: AuthService
// TDD Mode: Tests are skipped until function is implemented

import { describe, it, expect, vi } from 'vitest'
// Import will work once component skeleton exists
import { AuthService } from '../domain/auth-service'
import type { Credentials, AuthResult, AuthError } from '../types'

describe('AuthService', () => {
  // ═══════════════════════════════════════════════════════════
  // Function: validateCredentials
  // Status: SKIPPED (enable when implementing)
  // ═══════════════════════════════════════════════════════════
  describe.skip('validateCredentials', () => {
    it('validates correct email format', () => {
      // ... test implementation
    })

    it('rejects empty email', () => {
      // ... test implementation
    })
  })

  // ═══════════════════════════════════════════════════════════
  // Function: login
  // Status: SKIPPED (enable when implementing)
  // ═══════════════════════════════════════════════════════════
  describe.skip('login', () => {
    it('authenticates valid credentials', () => {
      // Arrange
      const credentials: Credentials = {
        email: 'user@example.com',
        password: 'validPassword123'
      }
      const authService = new AuthService()

      // Act
      const result = authService.login(credentials)

      // Assert
      expect(result).toBeInstanceOf(AuthResult)
    })

    it('returns AuthError for invalid password', () => {
      // ... test implementation
    })
  })

  // ... more skipped test blocks for each function
})
```

Write test file to TEST_FILE_PATH.
</step>

<step n="3" name="create_component_skeleton">
Create component file with empty skeleton.

**The skeleton has:**
- Class/struct declaration
- All function signatures (throwing "not implemented")
- Type imports
- No actual implementation

```typescript
// Example: TypeScript

// Generated by /opensdd:tdd-spec from spec.yaml
// Component: AuthService
// TDD Mode: Implement functions one-by-one as tests are enabled

import type { Credentials, AuthResult, AuthError, SessionId } from '../types'

export class AuthService {
  /**
   * Validate credentials format
   * TDD: Implement when validateCredentials tests are enabled
   */
  validateCredentials(credentials: Credentials): boolean {
    throw new Error('TDD: Not yet implemented')
  }

  /**
   * Authenticate user credentials and create session
   * TDD: Implement when login tests are enabled
   */
  login(credentials: Credentials): AuthResult | AuthError {
    throw new Error('TDD: Not yet implemented')
  }

  /**
   * Terminate user session
   * TDD: Implement when logout tests are enabled
   */
  logout(sessionId: SessionId): void {
    throw new Error('TDD: Not yet implemented')
  }
}
```

Write component file to COMPONENT_FILE_PATH.
</step>
